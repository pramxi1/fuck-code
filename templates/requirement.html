EXAMPLE DATA

- กรุณา import File ในรูปแบบ file.csv ( ตั้งชื่อไฟล์อะไรก็ได้ )
- รูปแบบของไฟล์ ต้องมี 2 คอลัมป์
- ส่วนหัวของคอลัมป์ที่ 1 ต้องชื่อว่า date
- ข้อมูลในคอลัมป์ date ต้องมีรูปแบบดังตัวอย่าง 1/1/2002
- ส่วนหัวของคอลัมป์ที่ 2 ต้องชื่อว่า sale
- ข้อมูลต้องมีอย่างน้อย 1 เดือน หรือ 30 เซลล์ ( Cell )
คำเตือน: ถ้าข้อมูลไม่ตรงตามข้อกำหนด ไฟล์จะไม่สามารถใช้งานได้และจะไม่สามารถพยากรณ์ได้
